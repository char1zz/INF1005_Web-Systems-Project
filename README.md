# SpinFit Final

Full-stack PHP + MySQL website for a Spin & HIIT studio.

## Included features
- Calendar-based class booking
- Booking cancellation with a 12-hour cutoff
- Login, sign up, profile, and membership pages
- Merchandise shop with separate product pages
- Product measurements / sizing details
- Cart, checkout, and order history
- Admin pages for classes, plans, orders, and products
- Premium hamburger / drawer menu inspired by modern retail layouts

## Quick setup
1. Copy `config/db.example.php` to `config/db.php`
2. Update your database credentials in `config/db.php`
3. Run `schema.sql`
4. Put the project in a folder named `spinfit` inside your local server root
5. Open `http://localhost/spinfit/`
